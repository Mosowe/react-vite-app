基于移动端的配置。

[vite官网](https://cn.vitejs.dev/guide/why.html)

优点：

> 1. 光速启动
> 2. 热模块替换
> 3. 按需编译

## 功能
    1. antd-mobile移动端组件
    2. axios网络数据交互
    3. hox状态管理
    4. react-router-dom路由管理
    5. postcss-px-to-viewport移动端px转vw/vh
    6. less预编译
    7. autoprefixer自动补全
    8. typescript语法

## 使用

1.  下载`git clone`
2.  `npm install`
3.  `npm run dev`
